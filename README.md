# Arun Portfolio

Framer portfolio hosted on Netlify via GitHub.

## Pages
- `index.html` — Homepage
- `mise/index.html` — Mise case study
- `fred/index.html` — FRED case study
- `promofit/index.html` — Promofit case study
- `stress-shield/index.html` — Stress Shield case study

## How to update

1. Export page source from Framer (right-click → View Page Source, then Save As)
2. Replace the relevant HTML file in this folder
3. Run:
```bash
git add .
git commit -m "update portfolio"
git push
```
Netlify will auto-deploy within ~30 seconds.
